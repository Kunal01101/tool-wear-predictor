# AI Tool Wear Predictor
Physics-based CNC tool wear prediction system for Mechanical Engineering Final Year Project.

## Models Used
- Taylor's Extended Tool Life Equation — V·Tⁿ·fᵃ·dᵇ = C
- Hastings (1980) Temperature Model
- Kienzle Cutting Force Model
- Trigger & Chao Wear Mechanism Classification
- ISO 3685 Failure Criterion (VB = 0.3 mm)

## Features
- Predicts Flank Wear (VB) in real time
- Remaining Useful Life (RUL) estimation
- Cutting force and interface temperature calculation
- Wear mechanism classification (Abrasive / Adhesive / Diffusion / Fatigue)
- Supports 5 tool materials × 6 workpiece combinations
- Works fully offline — no backend required
